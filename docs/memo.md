# Memo

## 構想

必須要件:

- 外部のサーバーにデータを格納する ( Firebase, Supabase 等を使う )
- n記一覧（ホーム）、日記作成、日記詳細、日記編集、日記検索 画面を作る
  - 日記一覧（ホーム）: 一覧表示に無限スクロールを使う
  - 日記検索: 日記一覧（ホーム）画面に内包しても専用の画面を作成してもどちらでも良い
- 日記は、アイコン、タイトル、本文、日付、画像を入力できるようにする
- 画像は任意入力、他は必須入力とする

## 方針

- 検索は、ホーム画面に内包する。
- UI/UX の方針
  - 最低限の操作で、目的を達成できるようにする
  - Apple ライクな見た目にする
  - 落ち着いたトーンで、親しみのあるテイストにする
- バックエンドは Firebase を使う。
- 開発は、 AI エージェントを使う
